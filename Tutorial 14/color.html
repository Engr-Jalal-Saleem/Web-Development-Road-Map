Let's learn about colors:

Colors can be represented in these ways:

  1. Color Keywords
  2. RGB
  3. Hexadecimal Color Code
  4. RGBA
  5. HSL

Color Keywords

Color keywords are the easiest way to represent colors in CSS. They are a set of predefined colors that can be used in CSS. Some of the color keywords are:

  • red
  • blue
  • green
  • yellow
  • black
  • white
  • orange
  • purple
  • pink
  • gray
  • brown

These are just a few of the color keywords. There are many more. You can find a list of all the color keywords here.

RGB

RGB stands for Red, Green, Blue. It is a way to represent colors in CSS. It is represented in the following way:

rgb(red, green, blue)

Each of the red, green, and blue values can be between 0 and 255. The red value represents the amount of red in the color. The green value represents the amount of green in the color. The blue value represents the amount of blue in the color. The higher the value, the more of that color is in the color. For example, rgb(255, 0, 0) is red. rgb(0, 255, 0) is green. rgb(0, 0, 255) is blue. rgb(255, 255, 255) is white. rgb(0, 0, 0) is black.

Hexadecimal Color Code

Hexadecimal color codes are another way to represent colors in CSS. They are represented in the following way:

#RRGGBB

The RR value represents the amount of red in the color. The GG value represents the amount of green in the color. The BB value represents the amount of blue in the color. The higher the value, the more of that color is in the color. For example, #FF0000 is red. #00FF00 is green. #0000FF is blue. #FFFFFF is white. #000000 is black.

RGBA

RGBA stands for Red, Green, Blue, Alpha. It is a way to represent colors in CSS. It is represented in the following way:

rgba(red, green, blue, alpha)

Each of the red, green, and blue values can be between 0 and 255. The red value represents the amount of red in the color. The green value represents the amount of green in the color. The blue value represents the amount of blue in the color. The higher the value, the more of that color is in the color. The alpha value represents the opacity of the color. The alpha value can be between 0 and 1. 0 means the color is completely transparent. 1 means the color is completely opaque. For example, rgba(255, 0, 0, 1) is red. rgba(0, 255, 0, 1) is green. rgba(0, 0, 255, 1) is blue. rgba(255, 255, 255, 1) is white. rgba(0, 0, 0, 1) is black.

HSL

HSL stands for Hue, Saturation, Lightness. It is a way to represent colors in CSS. It is represented in the following way:

hsl(hue, saturation, lightness)

The hue value can be between 0 and 360. The saturation value can be between 0% and 100%. The lightness value can be between 0% and 100%. For example, hsl(0, 100%, 50%) is red. hsl(120, 100%, 50%) is green. hsl(240, 100%, 50%) is blue. hsl(0, 0%, 100%) is white. hsl(0, 0%, 0%) is black.

Some of the color variables are:

  • --yellow: yellow;
  • --black: black;
  • --white: white;

  • --purple: purple;
